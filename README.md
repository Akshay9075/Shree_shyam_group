# Shree Shyam Groups Website

Modern multi-page static website for:
- Raj Clinic
- Shree Shyam Medistore
- Shree Shyam Builder & Developer

## Pages
- `index.html` (Home)
- `about.html` (About)
- `contact.html` (Contact)
- `raj-clinic.html` (Service)
- `shree-shyam-medistore.html` (Service)
- `shree-shyam-builder.html` (Service)

## Features
- Modern responsive navbar with services dropdown
- Sticky header with logo-aligned color palette
- Redesigned footer fixed at page bottom naturally via layout
- Attractive cards, icons, and gradient sections
- Scroll reveal animations and hover transitions
- Mobile menu support

## Run Locally
Open `index.html` in any modern browser.

For a better local dev experience, run a simple static server (optional).

### VS Code Live Server (optional)
Install Live Server extension and click **Go Live**.

### Python server (optional)
```powershell
cd "c:\Users\Akshay\OneDrive\Desktop\shree shyam groups"
python -m http.server 5500
```
Then open `http://localhost:5500`.
